# Название проекта
МегаПоиск


## Описание
Веб приложение для поиска и создания сайтов

## Функциональности
-Поиск сайтов
-Создание сайтов
-Регистрация и авторизация пользователей 

## Технологии

Перечислите технологии, которые использовались в проекте.

1. TypeScript
2. Express.js
3. PostgreSQL
4. Frontend (Next js)
